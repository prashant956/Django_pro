Selecting with jquery!

You can use the $ sign to select elements form the Dom It will feel a lot like selecting using jquery.

For example:

  • Selecting all a tags:
    • $('a')
  • Selecting all elements from the class 'container
    • $(".container")
  • Selecting all elements with id "special"
    • $("#special")
